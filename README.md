# MPLibrary
A Mario Party Library

## Building
 Requires this project https://github.com/KillzXGaming/STLibrary. Build with visual studio and add all the reference dlls which are in the same project folder.

## Supported Games
 - MP4-8
 - MPDS
 - MP10
